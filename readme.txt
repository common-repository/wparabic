WPArabic Make Possible Arabic writing in WordPress editor Arabi -> عربي

== Description ==

WP Arabic is simple and efficient Arabic writing plugin, this plugin will allow you to write Arabic with in the WordPress editors. When you will write roman Arabic like Text "Ana Ahb Alarbia" in editor It will convert that to "انا احب العربية". You can enable this option by clicking the button over editors.

<strong>Plugin Features</strong><br />

* Arabic Writing.
* Arabic Fonts (BalooBhaijaan, Scheherazade) and many more.
* Gutenberg Editor Block.
* Gutenberg Editor Sidebar Block Arabic Fonts (BalooBhaijaan, Scheherazade) and Font Size.


<strong>Upcoming Features</strong><br />
* Comments section Arabic Typing.
* Some Exiting hidden features.

== Installation ==

= Install from dashboard =
1. Visit 'Plugins > Add New'
2. Search for <strong>'WPArabic'</strong>x
3. Click on the 'Install Now'
4. Activate WPArabic from your Plugins page.
5. You will see Enable/Disable WPArabic button over WordPress editor.

= Manual upload =
1. Upload `/wparabic/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently asked questions ==
= I can't find the enable/disable option. =
*  You will see Enable/Disable WPArabic button over WordPress editor in post and page section.
